 Проекты

  1. simple_gpt — Реализация GPT-архитектуры с нуля

  Этот проект демонстрирует понимание архитектуры GPT. С использованием PyTorch реализованы все ключевые компоненты,
  включая Causal Self-Attention, MLP-блоки, позиционные эмбеддинги, а также полный цикл обучения. Это доказывает, что я не
  просто использую готовые фреймворки, а понимаю, как они работают "под капотом".

  2. pii_obfuscator — Система маскирования персональных данных
  Готовое к использованию приложение для маскирования персональных данных (PII) в текстах. Проект показывает реализацию двух стратегий дообучения (Full Fine-Tuning и Parameter-Efficient Fine-Tuning с LoRA). Система имеет модульную архитектуру, управляется через CLI и покрыта тестами.

  3. rag_service — Продвинутая "Вопрос-Ответная" система

  RAG-сервис, спроектированный как "инструмент" для интеграции в более крупные AI-агентные системы. Для достижения высокой точности и эффективности я реализовал инновационный двухэтапный пайплайн с Re-ranking'ом через Function
  Calling и многоуровневое семантическое кэширование.
